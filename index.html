Augmented Reality App


This app allows users to download and print image targets from my website, and when they look at them through the app, they can view videos,images, and slideshows stored on the website on top of the image targets. What's unique about this app (besides the fact that it's using augmented reality) is that the videos and images displayed in the app can be instantly changed by uploading new ones to the website.

This was my first major project. I used Vuforia augmented reality API, Unity3D, Android Studio, and Xcode to make this project. I learned a lot about how much it takes to start an app from scratch, design the UI, troubleshoot bugs, and then publish on both app stores (with Apple's very exhaustive review process). It's not currently on the iOS store, as I took it down to add more features that I'm currently working on.

This is just the beginning though, as I'm currently working on an app that's entirely cloud based, and allows the user to take images, type in a video URL, and then upload the image and URL to the cloud. Then anyone using the app can look at the image target and see what video or pictures the user has attached to the image target.
